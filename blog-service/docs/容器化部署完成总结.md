## 🛠️ 快速使用指南

```bash
# 从根目录执行
cd ..

# 构建并启动所有服务
docker-compose up -d --build

# 查看服务状态
docker-compose ps

# 查看应用日志
docker-compose logs -f blog-service

# 停止服务
docker-compose down
```
