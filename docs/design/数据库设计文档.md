# 数据库设计文档

## 1. 数据库概述

- **数据库类型**: MySQL 8.0+
- **字符集**: utf8mb4
- **排序规则**: utf8mb4_unicode_ci
- **逻辑删除**: 所有业务表使用 `deleted` 字段（0-未删除，1-已删除）
- **时间字段**: 所有业务表包含 `create_time` 和 `update_time`

## 2. 表结构设计

### 2.1 用户表 (user)

| 字段名 | 类型 | 说明 | 约束 |
|--------|------|------|------|
| id | BIGINT | 主键 | PK, AUTO_INCREMENT |
| username | VARCHAR(50) | 用户名 | UNIQUE, NOT NULL |
| email | VARCHAR(100) | 邮箱 | UNIQUE, NOT NULL |
| password | VARCHAR(255) | 密码（加密） | NOT NULL |
| nickname | VARCHAR(50) | 昵称 | |
| avatar | VARCHAR(255) | 头像URL | |
| role | VARCHAR(20) | 角色 | DEFAULT 'user' |
| status | TINYINT | 状态（0-禁用，1-启用） | DEFAULT 1 |
| last_login_time | DATETIME | 最后登录时间 | |
| last_login_ip | VARCHAR(50) | 最后登录IP | |
| deleted | TINYINT | 逻辑删除 | DEFAULT 0 |
| create_time | DATETIME | 创建时间 | NOT NULL |
| update_time | DATETIME | 更新时间 | NOT NULL |

**索引**:
- PRIMARY KEY (id)
- UNIQUE KEY uk_username (username)
- UNIQUE KEY uk_email (email)
- KEY idx_status (status)

## 3. 配置项说明

### 3.1 网站配置 (site.*)
- `site.name`: 网站名称
- `site.description`: 网站描述
- `site.logo`: 网站Logo
- `site.cover`: 网站封面
- `site.keywords`: 关键词
- `site.notice`: 网站通知

### 3.2 作者配置 (author.*)
- `author.name`: 作者名称
- `author.avatar`: 作者头像
- `author.bio`: 作者简介
- `author.social`: 社交信息（JSON）

### 3.3 关于配置 (about.*)
- `about.content`: 关于内容

### 3.4 隐私配置 (privacy.*)
- `privacy.show_ip`: 是否显示IP
- `privacy.show_location`: 是否显示地理位置
- `privacy.show_device`: 是否显示设备信息

### 3.5 通知配置 (notification.*)
- `notification.login_enable`: 登录通知开关
- `notification.comment_enable`: 评论通知开关
- `notification.comment_to_author`: 评论通知作者

### 3.6 邮件配置 (email.*)
- `email.smtp.host`: SMTP主机
- `email.smtp.port`: SMTP端口
- `email.smtp.username`: 用户名
- `email.smtp.password`: 密码
- `email.from`: 发件人
- `email.template.comment`: 评论通知模板
- `email.template.login`: 登录通知模板

### 3.7 日志配置 (log.*)
- `log.retention.days`: 日志保留天数（默认90天）
