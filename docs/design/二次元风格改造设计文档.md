## 🌸 三、樱花粒子系统设计

### 3.1 实现方案选择

**推荐方案**: CSS Particles + Framer Motion
- ✅ 性能优秀（CSS动画硬件加速）
- ✅ 易于控制和定制
- ✅ 响应式友好
- ❌ Canvas方案更复杂但灵活性更高（可选）

### 3.2 粒子属性配置

```typescript
interface SakuraParticle {
  id: string;
  size: number;           // 12-24px
  startX: number;         // 0-100vw
  delay: number;          // 0-15s
  duration: number;       // 7-15s
  rotationSpeed: number;  // 旋转速度
  swayAmount: number;     // 摆动幅度
  opacity: number;        // 0.6-0.9
}
```

### 3.3 粒子组件设计

**组件路径**: `blog-web/src/components/effects/SakuraRain.tsx`

```typescript
interface SakuraRainProps {
  count?: number;         // 粒子数量（默认 20）
  enabled?: boolean;      // 是否启用
  intensity?: 'light' | 'medium' | 'heavy';
}
```

### 3.4 性能优化策略
- 使用 `will-change: transform, opacity`
- 限制最大粒子数量（30-40个）
- 移动端减少粒子（15个）
- 使用 `contain: layout paint style`
- `prefers-reduced-motion` 禁用动画
