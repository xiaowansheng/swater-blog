# 后台管理系统设计文档

## 1. 系统概述

### 1.1 系统定位
Blog Admin 是 Swater Blog 的管理后台系统，提供完整的博客内容管理、用户权限管理、数据统计分析等功能。

### 1.2 系统目标
- 提供直观易用的管理界面
- 支持实时通知推送
- 丰富的数据可视化展示
- 完善的权限控制体系
- 响应式设计，支持多端访问

## 5. 路由设计

### 5.1 路由结构

```
/                    # 重定向到 /dashboard
/login               # 登录页
/dashboard           # 仪表盘
/article             # 文章列表
/article/edit/:id    # 文章编辑
/article/create      # 文章创建
/category            # 分类管理
/tag                 # 标签管理
/comment             # 评论管理
/talk                # 说说管理
/user                # 用户管理
/role                # 角色管理
/config              # 系统配置
/file                # 文件管理
/visitor             # 访客管理
/log/operation       # 操作日志
/log/error           # 异常日志
/friend-link         # 友链管理
/notification        # 通知管理
```

### 5.2 路由守卫
- 未登录用户重定向到登录页
- 已登录用户访问登录页重定向到仪表盘
- 根据用户权限动态显示菜单

## 6. 状态管理

### 6.1 认证状态 (auth.ts)
```typescript
interface AuthState {
  user: User | null
  token: string | null
  login: (username: string, password: string) => Promise<void>
  logout: () => Promise<void>
  getCurrentUser: () => Promise<void>
  isAuthenticated: () => boolean
}
```

### 6.2 通知状态 (notification.ts)
```typescript
interface NotificationState {
  notifications: Notification[]
  unreadCount: number
  addNotification: (notification: Notification) => void
  markAsRead: (id: number) => void
  markAllAsRead: () => void
  removeNotification: (id: number) => void
  clearNotifications: () => void
}
```

## 7. API集成

### 7.1 请求封装
- Axios实例配置
- 请求拦截器（Token注入）
- 响应拦截器（统一错误处理）
- 请求重试机制

### 7.2 接口规范
- 统一响应格式：`Result<T>`
- 分页响应格式：`PageResult<T>`
- 错误码处理
- 超时处理

### 7.3 接口分类
- 认证接口：`/api/auth/*`
- 管理接口：`/api/admin/*`
- 所有接口需要Token认证（除登录接口）

## 8. WebSocket集成

### 8.1 连接配置
- WebSocket地址：`ws://host/ws/notification`
- 连接建立时发送用户ID
- 自动重连机制
- 心跳检测

### 8.2 消息格式
```typescript
interface NotificationMessage {
  type: 'comment' | 'login' | 'system'
  title: string
  content: string
  timestamp: string
}
```

### 8.3 消息处理
- 接收消息后更新通知状态
- 显示通知提示
- 更新未读计数
- 播放提示音（可选）

## 9. UI组件设计

### 9.1 布局组件
- **BasicLayout**: 主布局（Header + Sidebar + Content）
- **Header**: 顶部导航（用户信息、通知、退出）
- **Sidebar**: 侧边栏菜单（可折叠）
- **Content**: 内容区域（面包屑 + 页面内容）

### 9.2 通用组件
- **PageHeader**: 页面头部（标题、操作按钮）
- **SearchForm**: 搜索表单（统一搜索样式）
- **DataTable**: 数据表格（分页、排序、筛选）
- **FormModal**: 表单弹窗（创建/编辑）

### 9.3 图表组件
- **LineChart**: 折线图（趋势数据）
- **BarChart**: 柱状图（对比数据）
- **PieChart**: 饼图（占比数据）
- **StatCard**: 统计卡片（数字展示）

### 9.4 编辑器组件
- **MarkdownEditor**: Markdown编辑器
  - 支持实时预览
  - 支持代码高亮
  - 支持图片上传
  - 支持工具栏快捷操作

## 10. 权限控制

### 10.1 路由权限
- 根据用户角色动态生成菜单
- 无权限路由自动隐藏
- 无权限访问时显示403页面

### 10.2 按钮权限
- 根据接口权限控制按钮显示
- 无权限操作时禁用按钮

### 10.3 数据权限
- 根据用户角色过滤数据
- 普通用户只能查看自己的数据

- 登出清除Token

### 16.2 数据安全

- XSS防护（输入过滤）

### 16.3 权限安全

- 后端权限验证（安全保障）
