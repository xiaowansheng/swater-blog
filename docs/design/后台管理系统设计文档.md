# 博客后台管理系统设计文档

## 1. 系统概述

博客后台管理系统是一个基于 React + TypeScript + Ant Design 构建的现代化单页应用，用于管理博客的内容、用户、系统配置等。

## 2. 技术架构

### 2.1 前端技术栈

| 技术 | 版本 | 用途 |
|------|------|------|
| React | 18.x | UI框架 |
| TypeScript | 5.x | 类型安全 |
| Ant Design | 5.x | UI组件库 |
| Tailwind CSS | 3.x | 样式工具 |
| Zustand | 4.x | 状态管理 |
| React Router | 6.x | 路由管理 |
| Axios | 1.x | HTTP客户端 |
| ECharts | 5.x | 图表可视化 |
| Vite | 5.x | 构建工具 |

### 2.2 后端API

后端基于 Spring Boot 3.x + MyBatis Plus 构建，提供 RESTful API。

API基础路径: `/api`

## 3. 功能模块

### 3.1 仪表盘 (Dashboard)
- 数据统计卡片展示
- 访问量趋势图
- 文章发布趋势图
- 设备/浏览器分布图

### 3.2 内容管理
- **文章管理**: CRUD、发布/下架、批量操作
- **分类管理**: 树形结构管理
- **标签管理**: 标签颜色自定义
- **说说管理**: 动态内容管理

### 3.3 互动管理
- **评论管理**: 审核、回复、删除
- **留言管理**: 审核、删除
- **友链管理**: CRUD

### 3.4 媒体管理
- **文件管理**: 上传、预览、删除
- **相册管理**: 相册和图片管理

### 3.5 系统管理
- **用户管理**: CRUD、角色分配、密码重置
- **角色管理**: CRUD、权限配置
- **菜单管理**: 树形菜单管理
- **系统配置**: 参数配置
- **关于页面**: Markdown编辑

### 3.6 监控管理
- **访客统计**: 访客列表、统计图表
- **操作日志**: 系统操作记录
- **异常日志**: 错误日志查看

### 3.7 通知管理
- 系统通知列表
- WebSocket实时推送

## 5. API对接规范

### 5.1 请求格式

```typescript
// 请求头
Authorization: Bearer <token>
Content-Type: application/json
```

### 5.2 响应格式

```typescript
interface Result<T> {
  code: number      // 状态码: 200成功, 401未授权, 其他错误
  message: string   // 消息
  data: T          // 数据
  timestamp: string // 时间戳
}

interface PageResult<T> {
  records: T[]     // 数据列表
  total: number    // 总数
  size: number     // 每页大小
  current: number  // 当前页
  pages: number    // 总页数
}
```

## 6. 状态管理

使用 Zustand 进行状态管理，主要包括：

- **auth**: 用户认证状态
- **notification**: 通知状态
- **tabs**: 标签页状态

## 8. 安全考虑

- JWT Token 认证
- 请求拦截器自动添加 Token
- 401 响应自动跳转登录
- 敏感操作二次确认

## 9. 性能优化

- 路由懒加载
- 组件按需加载
- 图片懒加载
- 请求防抖/节流
- 列表虚拟滚动（大数据量）
