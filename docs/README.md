## 维护约定

- 优先维护“当前生效”的文档，把一次性总结和历史方案放入 `90-提案与归档` 或模块内 `archive/`
- 模块强相关文档放在模块目录，根 `docs/README.md` 只负责建立统一入口
- 配置、端口、启动命令以仓库中的 `package.json`、`application*.yml`、`docker-compose*.yml` 为准
