# 配置说明文档

## 1. 配置文件结构

```
src/main/resources/
├── application.yml          # 主配置文件
├── application-dev.yml      # 开发环境配置
└── application-prod.yml     # 生产环境配置
```

## 2. 主配置文件 (application.yml)

```yaml
spring:
  profiles:
    active: dev  # 激活的环境配置

server:
  port: 8080

mybatis-plus:
  configuration:
    map-underscore-to-camel-case: true
  global-config:
    db-config:
      logic-delete-field: deleted
      logic-delete-value: 1
      logic-not-delete-value: 0
```

## 4. 生产环境配置 (application-prod.yml)

```yaml
    username: ${DB_USER:root}
    password: ${DB_PASSWORD}
    hikari:
      maximum-pool-size: 20
      minimum-idle: 10
```

### 4.2 Redis配置（使用环境变量）

```yaml
spring:
  data:
```

## 5. 环境变量说明

### 5.1 数据库环境变量
- `DB_HOST`: 数据库主机
- `DB_PORT`: 数据库端口
- `DB_NAME`: 数据库名称
- `DB_USER`: 数据库用户名
- `DB_PASSWORD`: 数据库密码

### 5.2 Redis环境变量
- `REDIS_HOST`: Redis主机
- `REDIS_PORT`: Redis端口
- `REDIS_PASSWORD`: Redis密码
- `REDIS_DATABASE`: Redis数据库编号

### 5.3 RabbitMQ环境变量
- `RABBITMQ_HOST`: RabbitMQ主机
- `RABBITMQ_PORT`: RabbitMQ端口
- `RABBITMQ_USER`: RabbitMQ用户名
- `RABBITMQ_PASSWORD`: RabbitMQ密码

### 5.4 Elasticsearch环境变量
- `ES_URIS`: Elasticsearch地址
- `ES_USERNAME`: Elasticsearch用户名
- `ES_PASSWORD`: Elasticsearch密码

### 5.5 OSS环境变量
- `OSS_ENDPOINT`: OSS端点
- `OSS_ACCESS_KEY_ID`: OSS AccessKey ID
- `OSS_ACCESS_KEY_SECRET`: OSS AccessKey Secret
- `OSS_BUCKET_NAME`: OSS Bucket名称
- `OSS_DOMAIN`: OSS域名

### 5.6 其他环境变量
- `FILE_STORAGE_TYPE`: 文件存储类型（local/oss）
- `AMAP_KEY`: 高德地图API Key

## 6. 配置优先级

1. 环境变量（最高优先级）
2. application-prod.yml / application-dev.yml
3. application.yml（最低优先级）

## 7. 配置验证

启动时会自动验证以下配置：
- 数据库连接
- Redis连接
- RabbitMQ连接
- Elasticsearch连接
- 邮件服务配置（如果启用）

配置错误会在启动时抛出异常并提示。
